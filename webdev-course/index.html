How to Test

  • Windows: Settings → Ease of Access → High contrast
  • macOS: System Preferences → Accessibility → Display → Increase contrast
  • Browser DevTools: Rendering tab → Emulate CSS media feature prefers-contrast

Expected Behavior

  • Normal: Standard colors and contrast
  • High Contrast: Black background, white text, strong borders
  • Low Contrast: Muted colors, softer appearance
